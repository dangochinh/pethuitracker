# Specs: Project Documentation Overview

This folder (`specs/`) will contain the individual specification files for the Pe Thui Tracker project once the documentation process (in `/opsx-apply`) is completed.

## Expected Specifications
- `api.md`: Serverless API routes mapped from `app/api/`.
- `pages.md`: Main routing logic and layout mappings.
- `components.md`: Reusable UI components and Modals.
- `core-logic.md`: Libraries, web-push utilities, google-sheets DB logic, and calculations.
